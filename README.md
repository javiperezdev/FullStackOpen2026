# Full Stack Open 2026

This repository contains my submissions and projects for the **[Full Stack Open](https://fullstackopen.com/en/)** course offered by the **University of Helsinki**.

The course is a deep dive into modern web development, focusing on the MERN stack (MongoDB, Express, React, Node.js), TypeScript, and CI/CD pipelines.

## Tech Stack & Skills
* **Frontend:** React, Redux, React Router, GraphQL, Apollo Client
* **Backend:** Node.js, Express, REST API
* **Database:** MongoDB (NoSQL), PostgreSQL (Relational)
* **Testing:** Jest, Cypress, React Testing Library
* **DevOps:** Docker, GitHub Actions (CI/CD)
* **Languages:** JavaScript (ES6+), TypeScript
